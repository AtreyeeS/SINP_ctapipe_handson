# SINP_ctapipe_handson
Notebooks for EHEPAP 2026, created by @shahjahanIqbal and @AtreyeeS for low level calibration and reconstruction of IACT data.

We will be using ctapipe version 0.29 (https://ctapipe.readthedocs.io/en/v0.29.0/index.html) with CTAO prod 5 simulations.

Please download the file "gamma-diffuse_with_images_00.dl2.h5" from https://zenodo.org/records/7298569"
